## What is Ubik?

Ubik is a hospitality platform built as a Progressive Web App (PWA) using Angular. It bridges two audiences:

- **Motel owners** — register establishments, add rooms with photos and amenities, set pricing, and monitor bookings from a centralized dashboard.
- **Guests** — discover motels by location using an interactive map, filter by availability and room type, and complete secure online reservations with Stripe payments.

## Platform Architecture

Ubik is built with a modern Angular stack:

| Technology | Role |
|---|---|
| Angular 20+ | Frontend framework with SSR support |
| TypeScript | Type-safe application code |
| Tailwind CSS v4 | Utility-first responsive styling |
| PrimeNG | UI component library |
| Leaflet | Interactive map rendering |
| Stripe.js | Client-side payment processing |
| RxJS | Reactive data streams |
| Cloudinary | Image upload and CDN delivery |
