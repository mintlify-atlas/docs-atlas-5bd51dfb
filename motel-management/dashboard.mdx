## Overview

The owner dashboard (`/dashboard/owner`) gives motel owners a high-level summary of their portfolio at a glance. It displays key operational metrics and provides quick-access actions to the most common management tasks.

## Accessing the dashboard

<Steps>
  <Step title="Log in as a motel owner">
    Use your owner credentials on the Ubik login screen. Only accounts with the `OWNER` role can reach the dashboard.
  </Step>
  <Step title="Navigate to /dashboard/owner">
    After logging in, go to `/dashboard/owner`. The route is protected by `permissionGuard`, which verifies your role before granting access. Unauthorized users are redirected away from this route.
  </Step>
</Steps>

<Warning>
  The `/dashboard/owner` route is protected by a `permissionGuard`. Attempting to access it without an authenticated owner session will redirect you to the login page.
</Warning>

## Quick actions

Below the metrics, the **Acciones** section provides two buttons:

| Button | Description |
|---|---|
| **Agregar Motel** | Opens the motel registration form at `/create-motel`. |
| **Ver Reservas** | Opens the reservations view for all your properties. |

## Component reference

The dashboard is implemented as a standalone Angular component:

```typescript
// src/app/views/dashboard/owner/owner.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-dashboard-owner',
  imports: [],
  templateUrl: './dashboard-owner.html',
})
export class DashboardOwner {}
```

`DashboardOwner` is intentionally lightweight — metrics are currently rendered as static values in the template. Dynamic data fetching from the API (using `MotelService.getMyMotels()` and reservation endpoints) is the expected next step in the component's development.
