Ubik organizes reusable UI into two directories:

- **`src/app/components/`** — feature-agnostic presentational components shared across pages.
- **`src/app/layout/`** — application-shell components that wrap every page (header, footer, mobile nav).

<Info>
All components are standalone Angular components (`standalone: true`). Import them directly into the `imports` array of any component or route that needs them — there is no shared module.
</Info>

## Layout components

The three layout components in `src/app/layout/` are included once in `app.component.html` and wrap all routed views.

**`app-header`** (`selector: 'app-header'`) renders the top navigation. It handles:
- Auth-aware links (login, profile, owner dashboard, admin dashboard)
- An autocomplete search bar (department → city → keyword)
- A sticky transparent nav that appears when scrolling up past 100 px on the home route

**`app-footer`** (`selector: 'app-footer'`) renders the branding footer. It has no inputs.

**`app-nav-bar-bottom`** (`selector: 'app-nav-bar-bottom'`) is the fixed bottom tab bar visible on mobile. It reacts to auth state and role to show the correct tabs, and collapses any open slide-up menu on route changes.
