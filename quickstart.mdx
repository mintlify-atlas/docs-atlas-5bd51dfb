<Tabs>
  <Tab title="As a guest">
    <Steps>
      <Step title="Visit the platform">
        Open Ubik in your browser. The home page loads at `/` and immediately shows you featured motels and a navigation bar.

        <Tip>Ubik is a PWA. On mobile, you can add it to your home screen for an app-like experience.</Tip>
      </Step>
      <Step title="Register as a customer">
        1. Tap **Sign up** or navigate to `/select-register`.
        2. Select **Customer** to go to the guest registration form at `/register-user`.
        3. Fill in your username, email, password, phone number, and date of birth.
        4. Submit the form. Your account is created with `roleId: 3` (customer).

        <Info>Already have an account? Go to `/login` and enter your username and password.</Info>
      </Step>
      <Step title="Explore motels">
        Navigate to `/explore`. The page shows an interactive Leaflet map centered on your current location alongside a list of available motels.

        Use the filter modal to narrow results by location or room type.
      </Step>
      <Step title="Pick a room">
        1. Click a motel card to open its profile at `/motelProfile/:id`.
        2. Browse available rooms — each card shows the room type, amenities, and pricing.
        3. Click a room to view its full detail page at `/roominfo`.
      </Step>
      <Step title="Make a reservation">
        On the room detail page, select your check-in and check-out dates and confirm the booking details.
      </Step>
      <Step title="Complete payment">
        Ubik uses Stripe for secure payment processing.

        1. The payment modal opens with your reservation summary.
        2. Enter your card details in the Stripe-powered form.
        3. On success, you are redirected to `/payment/success`. If the payment fails, you land on `/payment/failure` and can retry.
      </Step>
    </Steps>
  </Tab>
  <Tab title="As a motel owner">
    <Steps>
      <Step title="Register as an owner">
        1. Navigate to `/select-register`.
        2. Select **Owner** to go to the owner registration form at `/register-owner`.
        3. Complete your account details. Owner accounts receive a different role ID from customer accounts, granting access to owner-specific routes.
      </Step>
      <Step title="Create your motel profile">
        1. After logging in, navigate to `/create-motel`.
        2. Enter your establishment name, description, address, and location coordinates.
        3. Upload your motel's cover image — files are sent to Cloudinary (cloud name: `du4tcug9q`, folder: `Ubik`) and the returned URL is saved with your motel record.
        4. Submit to register your property on the platform.
      </Step>
      <Step title="Add rooms with images">
        1. Go to `/create-room` from your motel's management page.
        2. Define the room type, capacity, pricing, and list of amenities.
        3. Upload room photos via the Cloudinary integration.
        4. Save each room. Repeat for all room types in your establishment.
      </Step>
      <Step title="Manage reservations from the dashboard">
        Navigate to `/dashboard/owner` to view all reservations guests have made at your properties.

        <Note>The owner dashboard is protected by `permissionGuard('view_dashboard')`. You must be logged in as an owner to access it.</Note>
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Developer setup

<Steps>
  <Step title="Prerequisites">
    Ensure you have the following installed before proceeding:

    - **Node.js** v18 or later
    - **pnpm** (the project uses `pnpm@10.32.1` as its package manager)
    - **Angular CLI** — install globally if you don't have it:

    ```bash
    npm install -g @angular/cli
    ```
  </Step>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Juankos0714/Ubik-App.git
    cd Ubik-App
    ```
  </Step>
  <Step title="Install dependencies">
    <CodeGroup>

    ```bash pnpm
    pnpm install
    ```

    ```bash npm
    npm install
    ```

    </CodeGroup>
  </Step>
  <Step title="Run the development server">
    <CodeGroup>

    ```bash ng
    ng serve
    ```

    ```bash npm
    npm start
    ```

    </CodeGroup>

    Both commands run `ng serve` as defined in `package.json`.
  </Step>
  <Step title="Open the app">
    Open your browser and navigate to:

    ```
    http://localhost:4200
    ```

    The app hot-reloads automatically as you edit source files.

    <Tip>The API base URL is configured in `src/environments/environment.ts`. In development it points to `https://ubik-back.duckdns.org/api`.</Tip>
  </Step>
</Steps>

## Useful commands

| Command | Description |
|---|---|
| `ng serve` | Start the development server at `localhost:4200` |
| `npm start` | Alias for `ng serve` |
| `ng build` | Build the application for production |
| `ng test` | Run unit tests with Karma |
| `ng build --watch --configuration development` | Rebuild on file changes |
